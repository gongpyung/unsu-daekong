# 🫘 운수대콩 (Unsu Daekong) - 로또 번호 생성기

**운수대콩**은 귀여운 디자인과 강력한 기능을 갖춘 웹 기반 로또 번호 생성 애플리케이션입니다.
사용자 친화적인 인터페이스와 다양한 생성 모드를 통해 나만의 행운의 번호를 만들어보세요!

![운수대콩 로고](unsu-daekong.png)

## ✨ 주요 기능 (Key Features)

### 1. 🎲 다양한 번호 생성 모드
-   **Random (랜덤)**: 완전 무작위로 번호를 생성합니다.
-   **Hot (뜨거운 번호)**: 역대 당첨 번호 중 가장 많이 나온 번호 위주로 생성합니다. (가중치 적용)
-   **Cold (차가운 번호)**: 역대 당첨 번호 중 가장 적게 나온 번호 위주로 생성합니다.

### 2. ⚙️ 고급 설정 (Advanced Settings)
-   **포함할 번호**: 반드시 포함하고 싶은 번호를 지정할 수 있습니다.
-   **제외할 번호**: 절대 나오지 않게 하고 싶은 번호를 지정할 수 있습니다.
-   **유연한 입력**: 쉼표(`,`) 또는 띄어쓰기(` `)로 번호를 쉽게 구분하여 입력할 수 있습니다.

### 3. 📜 생성 기록 (History)
-   생성된 번호는 자동으로 기록됩니다.
-   각 기록을 개별적으로 삭제하거나, 전체 기록을 한 번에 삭제할 수 있습니다.
-   기록은 브라우저에 저장되어 새로고침 후에도 유지됩니다.

### 4. 🎨 테마 및 애니메이션
-   **다크/라이트 모드**: 눈이 편안한 다크 모드와 깔끔한 라이트 모드를 지원합니다.
-   **애니메이션 토글**: 번호 생성 시의 화려한 애니메이션을 켜거나 끌 수 있습니다.
-   **반응형 디자인**: PC, 태블릿, 모바일 등 모든 기기에서 최적화된 화면을 제공합니다.

## 🛠️ 기술 스택 (Tech Stack)
-   **HTML5**: 시맨틱 마크업 구조
-   **CSS3**: Flexbox, Grid, CSS Variables, Animations, Glassmorphism 효과
-   **JavaScript (Vanilla JS)**: 로컬 스토리지 관리, DOM 조작, 알고리즘 구현

## 🚀 설치 및 실행 (How to Run)
이 프로젝트는 정적 웹 페이지로 구성되어 있어 별도의 서버 설치가 필요 없습니다.

1.  이 저장소를 클론(Clone)하거나 다운로드합니다.
2.  `index.html` 파일을 브라우저에서 엽니다.
3.  바로 사용 가능합니다!
