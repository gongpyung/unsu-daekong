# Changelog

## [Unreleased]

### Added
- 역대 당첨번호 기반 핫/콜드 번호 빈도 분석 (`calculateFrequencies`)
- 역대 당첨번호와 동일 조합 중복 방지 (Set 기반 O(1) 체크)
- 당첨번호 데이터 자동 수집 스크립트 (`npm run fetch-data`)
- 롤링 애니메이션 (슬롯머신 효과, 번호별 순차 정지)
- 즉시 뽑기 모드 (애니메이션 토글)
- 기록 전체 삭제 기능 (2단계 확인)
- 핵심 로직 테스트 코드 (Vitest, 17개 테스트)
- CLAUDE.md 프로젝트 가이드

### Changed
- 핫/콜드 번호를 하드코딩에서 실제 당첨 데이터 기반으로 변경
- 포함/제외 번호 입력을 공백 구분으로 통일
- 모드 선택 버튼 동일 크기로 정규화
- 브라우저 탭 제목을 "loddo"로 변경
- 우측 상단 테마/애니메이션 버튼 세로 배치
- 고급 설정 제목 가운데 정렬
- 최근 기록 제목 가운데 정렬

## [0.1.0] - 초기 버전

### Added
- 랜덤/핫번호/콜드번호 3가지 모드 번호 생성
- 포함/제외 번호 고급 설정
- 다크 모드 지원
- 최근 기록 저장 (localStorage, 최대 20개)
- 개별 기록 삭제 (슬라이드 아웃 애니메이션)
- 반응형 모바일 레이아웃
- 플로팅 파티클 배경 효과
