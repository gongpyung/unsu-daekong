

# 🎲 운수대쿵 - 귀엽뽀짝 로또 번호 생성기

로또 번호 생성기를 귀여운 애니메이션과 파스텔톤 디자인으로 만듭니다.

## 디자인 컨셉
- **파스텔 컬러 팔레트**: 연핑크, 연보라, 연노랑 등 부드러운 색상
- **둥글둥글한 UI**: 카드, 버튼 모두 큰 border-radius
- **귀여운 이모지 & 아이콘** 활용
- **뽀짝 애니메이션**: 통통 튀기는 bounce, 흔들흔들 wiggle, 반짝반짝 sparkle 효과

## 주요 기능 & 화면

### 1. 메인 페이지
- 귀여운 타이틀 영역 (통통 바운스 애니메이션)
- 번호 표시 영역: 로또 공이 하나씩 통통 튀어나오는 애니메이션
- 각 공은 번호 범위별로 다른 파스텔 색상 (1~10 핑크, 11~20 노랑 등)

### 2. 생성 모드 탭 (Random / Hot / Cold)
- 탭 전환 시 부드러운 슬라이드 애니메이션
- 선택된 탭은 살짝 커지고 그림자가 생기는 효과
- Random: 완전 랜덤 번호 생성
- Hot: 자주 나오는 번호 위주 (시뮬레이션)
- Cold: 안 나오는 번호 위주 (시뮬레이션)

### 3. 번호 생성 버튼
- 큰 둥근 버튼, 누르면 흔들흔들 + 반짝 효과
- 생성 중 로딩: 공이 빙글빙글 도는 귀여운 애니메이션
- 번호가 하나씩 팡! 하고 나타남 (staggered pop-in)

### 4. 고급 설정 (포함/제외 번호)
- 아코디언으로 열리는 설정 패널
- 번호 입력 필드

### 5. 최근 생성 기록
- 생성된 번호 세트가 카드로 쌓이는 애니메이션
- 기록 삭제 시 카드가 사르르 사라지는 효과
- localStorage에 기록 저장

### 6. 뽀짝 애니메이션 모음
- **bounce**: 공이 통통 튀는 효과
- **wiggle**: 버튼/아이콘이 좌우로 흔들리는 효과  
- **sparkle**: 반짝이는 별 파티클 효과
- **pop-in**: 요소가 팡! 하고 나타나는 효과
- **float**: 배경 요소가 둥실둥실 떠다니는 효과
- 배경에 떠다니는 귀여운 별/하트/클로버 파티클
